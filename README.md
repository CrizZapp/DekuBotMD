## 🛠️ Instalación en Termux

Para instalar y ejecutar el bot en tu dispositivo, sigue estos comandos paso a paso:

1. **Actualiza los paquetes:**
pkg update && pkg upgrade -y

2. **Instala las dependencias necesarias:**
pkg install git nodejs ffmpeg imagemagick -y

3. **Clona el repositorio:**
git clone https://github.com/CrizZapp/DekuBotMD
cd DekuBotMD

4. **Instala los módulos de Node.js:**
npm install

5. **Inicia el proyecto:**
npm start

## ⚡ DekuBotMD

Repositorio oficial de **DekuBotMD**, un bot multipropósito para WhatsApp desarrollado en Node.js.

---

## 📥 Instalación

### ☁️ Clonar Repositorio

```bash
git clone https://github.com/CrizZapp/DekuBotMD
```

### 📂 Entrar a la carpeta

```bash
cd DekuBotMD
```

### 📦 Instalar dependencias

```bash
npm install
```

### ▶️ Iniciar el bot

```bash
npm start
```
